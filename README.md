# Customer Value Prediction

This project predicts high‑value customers from an online retail dataset using three ML classifiers (Logistic Regression, Decision Tree, KNN).

## How to run

1. Clone this repository.
2. Place `customer_retail.csv` in the `data/` folder (or in the root directory).
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   
   python main.py
